=== 1 ===
Название: less
Описание: Пошаговый просмотр содержимого файлов
Синтаксис: less <файл>
Пример: less /var/log/syslog
Примечание: Удобнее cat для больших файлов
=== 2 ===
Название: grep
Описание: Поиск текста по шаблону в файлах
Синтаксис: grep [опции] "шаблон" файл
Пример: grep "error" log.txt
Примечание: Часто используется с | (пайпом)
=== 3 ===
Название: find
Описание: Поиск файлов и каталогов
Синтаксис: find путь [условия]
Пример: find /home -name "*.txt"
Примечание: Очень мощный инструмент для поиска
=== 4 ===
Название: df
Описание: Информация о дисковом пространстве
Синтаксис: df [опции]
Пример: df -h
Примечание: С -h отображает в удобных единицах (GB, MB)
=== 5 ===
Название: scp
Описание: Копирование файлов по SSH
Синтаксис: scp источник назначение
Пример: scp file.txt user@server:/home/user/
Примечание: Используется для удалённой передачи файлов
=== 6 ===
Название: dmesg
Описание: Вывод сообщений ядра Linux
Синтаксис: dmesg [опции]
Пример: dmesg | grep usb
Примечание: Полезно для диагностики оборудования
=== 7 ===
Название: tmux
Описание: Мультиплексор терминала (разделение экрана, фоновые сессии)
Синтаксис: tmux [опции]
Пример: tmux new -s mysession
Примечание: Удобен для работы на серверах по SSH
=== 8 ===
Название: apropos
Описание: Поиск команд и страниц man по ключевому слову
Синтаксис: apropos ключевое_слово
Пример: apropos network
Примечание: Полезно, если не помнишь точное название команды
=== 9 ===
Название: awk
Описание: Язык обработки текстовых данных и файлов
Синтаксис: awk 'условие {действие}' файл
Пример: awk '{print $1}' file.txt
Примечание: Мощный инструмент для парсинга и анализа данных
=== 10 ===
Название: at
Описание: Запуск задач по расписанию (однократно)
Синтаксис: at время
Пример: echo "echo hello > /tmp/test.txt" | at now + 1 minute
Примечание: Альтернатива cron, но для разовых задач
